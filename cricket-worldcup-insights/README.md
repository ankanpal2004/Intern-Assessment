# Cricket World Cup Insights
Cricket World Cup Insights is a Business Intelligence project that visualizes
historical Cricket World Cup data to uncover performance trends and insights.

## Tool Used
- Google Looker Studio

## Data Source
- Cricket World Cup data from public sources (Wikipedia / ESPN Cricinfo)

## Dashboard Features
- Country-wise performance filters
- Player statistics and comparisons
- Trends of highest runs in a single innings
- Venue-based performance analysis
- Geospatial visualization for Indian team venues

## How to View
1. Open the Google Looker Studio report link
2. Use filters to explore team and player performance
3. Analyze trends and comparisons visually

## Output
An interactive dashboard showing key Cricket World Cup insights
